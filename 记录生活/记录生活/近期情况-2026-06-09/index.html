关于博客

看了一眼，上次更新技术文章已经是三个月前了，好像是第一次这么久都没有写。

后面大概会几篇关于微服务的文章，最近应该会开始着手写关于BFF的。在我大概再进行一些实践更加理解它时，或者说在尝试出一些比较好的实践时。
之前的读书笔记，那本《hello agents》大概只看了一半，主要是一些概念，后面一半主要是实践。
概念的部分也许不会有多大变化，但实践的部分过去了这么久，大概率已经有比较新的实践方式了。
中断是因为工作，后面也不知道还会不会捡起那本读了一半的书去尝试剩下的实践了。

不过最近托朋友的福，爽爽的用了段时间的 GPT 5.5。
codex 和 claude 这些 agent 的实践确实不错的，加上一些 mcp 和 skills，再加上 plan mode 之类的，确实是可以完全 vibe coding 了，完全不用自己古法编程。
这么看来 AI 的发展还是很快的，两年多前刚体验 cursor 时，算是第一次尝试完整的前端工程项目。
生成了十几个文件就开始乏力了，在比较系统的工程上效果是完全达不到预期的。但今天在这些模型和 agent 的加持下，上千个文件，几十万行到百万行代码量级的项目下，模型也可以驾驭得住。
不过目前 token 的消耗量是非常恐怖的，价格也是非常之高。
总之，AI 的应用我觉得还有很大的空间，目前的限制主要还是算力，也就是价格。

关于工作

前面有说到这么久没更是因为工作，其实之前更新比较频繁也是跟工作有些关系。
更新频繁大概是因为在想要换工作的前夕，很久没更就是换了工作进入新的环境，需要适应。

这次跳槽是在三月底。从准备简历，到面试，再到拿到 offer，就花了一个星期。然后提离职，再到体检、入职又花了一个星期。整体就是非常的丝滑。
跳槽的原因也是比较简单，上家是比较草台的，开发总共也就六七人。干着把项目从 php 迁移 go 的工作，连需求文档都没有，甚至开发直接对接客户。
直到核心 php 提离职（在我提之前的一周左右），我就意识到该跑路了。
听前同事后来的叙述，几个开发都和新公司（客户公司）重新签了合同，大概是连着业务带团队都打包给了客户。应该会有大刀阔斧的改革，不然很难做出成果。

那时候简历大概准备了两天，投了三四天。有三个面试
第一个是线上的面试，比较离谱，过程很不愉快，草草的就结束了。业务是社区团购相关，还有物联网相关的产品。
第二个是电话的面试，聊了应该有二三十分钟，聊的一般，感觉在转型探索新业务。业务是硬件产品，客户是政企，不赚钱发展新业务，直播矩阵相关，额外问了爬虫。（这个单休且有固定加班，应该是要高了，后续没回复）
第三个是线下的面试，一次性面了三轮，应该有两个多小时。主要的技术面试官（现在的leader）非常的和蔼，属于引导型的面试了。
除了一些基础的问题外，还会有些场景问题，场景会一点点增加变化，引导你进行不同的设计，以应对不断变化的场景。这场面试应该是我目前为止所有面试中面的最舒服的一场了。

offer 发的也很快，3月19号面试完，20号就发了 offer。4月1号就入职了。还是愚人节，hhh
新公司给配 macbook （给我是台 m1 的 macbook pro，虽然是老电脑，但 32G 内存也是很爽了）
用飞书，每天有早会，每月有次需求评审的大会。所有的需求都有需求文档，对应开发还需要写技术方案文档，方案过会要review通过之后才进行开发。
相对应的还有测试和发布的流程，提测、提PR（代码会有leader或其他人review）、发布准备的文档等。部署也有完整的 jenkins 流水线，kubernetes 集群等。
从这些能看出来，工作节奏还是非常快的，我还是很不适应的，两个月过去也还只是稍微好点，感觉时间过的非常之快。

另外就是身边朋友不断有人被裁，能看出整个环境非常的不好。能找到一份还满意的工作不错了。鸽鸽说找工作就是抽奖，非常看运气。
第一个是大学同学，比我晚一个月来杭州。也是原来实习公司被裁了才被同时推到杭州新公司来的，年后刚去上班就被干掉了。也是来我这过渡了一个多月，回家复习准备考研去了。
第二个是南京的前同事，还是我第一份工作的同事。也是年后被裁，在南京找了两月没找到，来了杭州。正好我换工作，租的房也差不多到期。现在和他整租了一套，感觉非常好。（公司离得挺近，也是很难得了）
第三个算是启蒙导师了，五一还去线下专门见了他。结果五月下旬就被裁了，新工作单休还没之前工资高，也是非常难受了。二胎压力还是有点大。
第四个也是大学同学，裸辞的，新的工作不知道找到没。
第五个是完全没有想到的，技术经理职位算是管理层了。我认为会相对比较稳定的，也是在我意料之外了。
还有两个不是相关行业的，一个也是受不了裸辞了，不过他好像三月多才入职的来着；另一个也是 gap 17 个月之后终于找到了份行政的工作，今天第一天上班（看起来是一个人干三个人的活，也不知道能坚持多久）。

End

已经干了两个月了，再干四个月，先度过六个月的试用期吧。
希望这家我能干的长久一点，虽然项目的代码很烂，但我觉得这是多人协作不可能避免的问题。干的大部分又是迁移重构的活，理别人代码还是很难受的。

其他时间目前都在用 AI 探索微服务相关的东西，想写出自己的一些实践。
后面随着 OpenAI 限制越来越严格，账号越来越难整，应该会多些时间写些博客吧。
